X Country Filter — How to Use

What X Country Filter Does

X Country Filter lets you control whose posts you see on X (Twitter) based on the geographic origin of their account. It filters posts in your feed, replies, search results, notifications, and suggestions.

Privacy first: All processing happens locally in your browser. No data is ever sent to external servers (unless you opt into the online location lookup).

Filter Modes

Blocklist Mode

Select countries you want to hide. Posts from accounts in those countries are filtered out. Everything else is shown normally.

Best for: Hiding spam from specific regions while keeping your feed mostly unchanged.

Allowlist Mode

Select countries you want to keep. Only posts from accounts in those countries are shown. Everything else is hidden.

Best for: Creating a focused feed from trusted regions. More aggressive — use with care.

How Country Detection Works

The extension reads profile information that X already loads for each account and uses several methods to determine the account's country:

  1. Flag emoji — If the profile location contains a country flag emoji (e.g. 🇧🇷), the country is identified from it.
  2. Country name — The location text is matched against known country names and common aliases (e.g. "USA", "America", "UK").
  3. City name — Major world cities are mapped to their countries (e.g. "Lagos" → Nigeria, "Mumbai" → India).
  4. Partial match — If the location text contains a country name anywhere (e.g. "Living in Brazil"), it's matched.
  5. Language — For languages spoken in only one country (e.g. Japanese → Japan), the account language is used as a signal.
  6. Online lookup (optional) — If enabled, unrecognized locations are sent to OpenStreetMap's Nominatim service for resolution.

Accounts with no location set are labeled "unknown." You can choose to show or hide them in settings.

Display Modes

Completely Hidden

Filtered posts are removed entirely from the page. You won't see any trace of them.

Collapsed with Reveal

A small bar replaces each filtered post showing the detected country. Click "Show" to reveal the original post.

Changing Your Settings

Click the X Country Filter icon in your browser toolbar to open the settings popup. From there you can:

  • Toggle ON/OFF — Enable or disable filtering instantly.
  • Switch mode — Change between Blocklist and Allowlist.
  • Change display — Switch between Hidden and Collapsed.
  • Select countries — Search and check/uncheck countries.
  • Hide unknowns — Choose whether to filter accounts with no detected country.
  • Online lookup — Enable the Nominatim API for better detection of ambiguous locations.
  • Reset — Clear all settings and restart onboarding.

Changes take effect immediately — no page refresh needed.

Tips & Limitations

  • Detection works best on accounts that have a location set in their profile.
  • Accounts with no location or a fake location may not be accurately detected.
  • The user cache refreshes every 7 days. If someone updates their location, the change will be picked up after their cache entry expires.
  • If X updates their internal code, the extension may need an update to keep working.
  • The online lookup (Nominatim) sends location strings to an external service. It's off by default for privacy.
